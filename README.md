# SeniorProject
### how to install??

1. install nodejs yourself
2. clone this respository
3. `npm install`

### how to run??

`npx webpack`
or
`npm run build`

### development tricks

1. `npm run start:dev`
2. open another terminal 
3. `npm run build`
4. go to http://localhost:8080 , find the dist/index.html for live server
